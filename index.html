POST /tasks/form

Post to tempForm. Return response with parsed task with title, start time, end time and clashedTask (optional).

The temp tasks to create. Provide timing, date and description in single string to be parsed.

nlp: string
sequentialTasks: object[]
jwt

signed user token saved in cookie that would decrypt to user id

 cookie string
Origin

only 4200 allowed for now

 header string
200 OK

Successful response could be singleData, sequentialData or recurringData with clashedTask (could be empty object)

Example for application/json
{
    "clashedTask":
      {
        "title": "swimming",
        "scheduledStartDateTime": "",
        "scheduledEndDateTime": "",
      },
    "singleData":
      {
        "title": "swimming",
        "scheduledStartDateTime": "",
        "scheduledEndDateTime": "",
      },
    "sequentialData":
     [
      {
        "title": "swimming",
        "scheduledStartDateTime": "",
        "scheduledEndDateTime": "",
      },
      {
        "title": "swimming",
        "scheduledStartDateTime": "",
        "scheduledEndDateTime": "",
      }
    ],
    "recurringData":
     [
      {
        "title": "swimming",
        "scheduledStartDateTime": "",
        "scheduledEndDateTime": "",
      },
      {
        "title": "swimming",
        "scheduledStartDateTime": "",
        "scheduledEndDateTime": "",
      }
    ]
  }
400 Bad Request

User input does not include valid start and end times.

Example for application/json
{
  "code": "INVALID_DATETIME",
  "message": "pls input valid start and end times"
}
